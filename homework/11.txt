The purpose of this homework would be to try to use virtual environments and uv to create an isolated project.

1. Install uv from https://docs.astral.sh/uv/getting-started/installation/
2. Create an isolated folder for python project.
3. in that folder write a command (after opening that folder in vscode): 'uv init .'
4. in vscode powershell '.\.venv\Scripts\activate'
5. uv add langchain langchain-openai langchain_chroma langchain_community langchainhub beautifulsoup4 rich
6. Setup .env file.
7. Copy code from Generative-AI-Python-2025-05\samples\jupyter-notebook\rag.py
and try running.
